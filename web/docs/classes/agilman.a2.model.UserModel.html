createUser()

createUser() : array

Creates a new user.

Checks to see if user email exists. If so, returns user simply as false. Validates form entry. Any errors are returned as info. If there are no errors from the form data, it then saves locally. Password is hashed so only hashed password is saved. Current user is retrieved to return. Any errors from form validation are returned as info.

Returns

array —
  • on success (user details / no error) or errors (error info)
